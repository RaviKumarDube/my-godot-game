# 🎮 2D Platformer Game (Godot)

A simple 2D platformer built using Godot Engine as part of my learning journey.

---

## 🚀 Features

* Player movement (left, right, jump)
* Death zone system
* Game Over screen
* Restart button

---

## 🎮 Controls

* Move: Arrow keys
* Jump: Space / Enter
* Restart: Button click

---

## 🛠️ Built With

* Godot Engine
* GDScript

---

## 📂 Structure

* Player
* Kill Zone
* UI (Label + Restart Button)
