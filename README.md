# my-godot-game
A simple my first 2D game where a player move in a linear path in map when player collide with ground then game over, see a restart button at the top to restart the game.


#  2D Platform Game (Godot)

A simple 2D platform built using Godot Engine as part of my learning journey. I learning something new and i made this simple 2d game as first game developer.

---

##  Features

* Player movement (left, right, jump)
* Death zone system
* Game Over screen
* Restart button

---

## Controls

* Move:-Arrow keys
* Jump:-Space / Enter
* Restart:-Button click

---

##  Built With

* Godot Engine
* GDScript

---

##  Structure

* Player
* Kill Zone
* UI (Label + Restart Button)
